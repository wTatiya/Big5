# คู่มือพัฒนาเครื่องมือประเมิน DISC สำหรับพยาบาลไทย

## บทนำ

คู่มือนี้ออกแบบเพื่อพัฒนาแบบประเมิน DISC สำหรับพยาบาลไทย โดยเน้นความเหมาะสมทางวัฒนธรรม การใช้งานจริงในเวิร์กช็อป และความเข้าใจง่ายสำหรับผู้ทำแบบประเมิน ทั้งในรูปแบบออนไลน์และเอกสารกระดาษ

## แบบสอบถามฉบับสั้น (24 ข้อ)

### คำชี้แจงผู้ทำแบบประเมิน

เลือกคำตอบเพียง 1 ข้อในแต่ละคำถามที่ “ใกล้เคียงคุณที่สุด” ในสถานการณ์ทั่วไป (ไม่ใช่วันที่เครียดสุดหรือดีสุด) ไม่มีถูกผิด

### แบบประเมิน (ผู้ทำเห็นเฉพาะส่วนนี้)

#### **หมวดการทำงานและแก้ปัญหา**

1. เริ่มเวรแล้วงานค้างเยอะ คุณมักจะ…
   - A ตรวจสอบรายละเอียดงานค้างทั้งหมดก่อนเพื่อวางแผนให้แม่นยำ
   - B ชวนคุยทีมให้ฮึกเหิม แล้วเริ่มลงมือแบบ “ไปด้วยกัน”
   - C กำหนดเป้าหมายและจัดลำดับงานเร่งด่วนทันที
   - D เข้าช่วยเหลือและสนับสนุนเพื่อนเพื่อให้งานลื่นไหล ลดความตึงเครียด
2. เมื่อเกิดเหตุฉุกเฉิน (CPR) บทบาทที่คุณถนัดคือ…
   - A ผู้จดบันทึกและตรวจสอบขนาดยา/เวลา (Recorder/Timekeeper) อย่างแม่นยำ
   - B ผู้ช่วยที่นิ่งสงบ คอยส่งเครื่องมือและดูแลความเรียบร้อย
   - C ผู้นำสั่งการ (Team Leader) ตัดสินใจฉับไว
   - D ผู้ประสานงาน สื่อสารกับญาติและทีมด้วยความกระตือรือร้น
3. เมื่อต้องเรียนรู้อุปกรณ์การแพทย์ใหม่…
   - A รอให้คนอื่นใช้จนชินก่อน แล้วค่อยเรียนรู้ตามขั้นตอนอย่างช้าๆ
   - B อ่านคู่มืออย่างละเอียด ศึกษาข้อมูลเทคนิคก่อนเริ่มใช้
   - C ลองจับลองทำเลย อยากรู้การทำงานเพื่อให้ใช้ได้เร็วที่สุด
   - D ชอบเรียนรู้ไปพร้อมๆ กับเพื่อนๆ ในวอร์ด ให้คนอื่นช่วยสอน
4. เมื่อแพทย์เขียนคำสั่งการรักษา (Doctor's order) ไม่ชัดเจน…
   - A ถามเพื่อนร่วมงานว่า "หมอคนนี้ปกติเขียนแบบนี้หมายความว่าไง?"
   - B ตรวจสอบประวัติเก่าและแนวทาง (Protocol) ก่อน แล้วค่อยถามเพื่อความชัวร์
   - C โทรหาแพทย์ทันทีเพื่อถามให้รู้เรื่อง กันความล่าช้า
   - D พยายามแกะลายมือเองก่อน หรือรอจังหวะหมออารมณ์ดีค่อยถาม
5. สไตล์การส่งเวร (Handoff) ของคุณ…
   - A เล่าเรื่องราว มีสีสัน แลกเปลี่ยนความเห็นกับเพื่อนร่วมเวร
   - B รวดเร็ว กระชับ เน้นประเด็นสำคัญและแผนที่ต้องทำต่อ
   - C เป็นระเบียบ เรียงลำดับข้อมูลครบถ้วน อ้างอิงผล Lab เป๊ะๆ
   - D พูดด้วยน้ำเสียงนุ่มนวล ใส่ใจรายละเอียดความรู้สึกคนไข้
6. สิ่งที่คุณไม่ชอบที่สุดในเวลางาน…
   - A บรรยากาศเงียบเหงา หรือถูกตัดบทเวลาพูด
   - B ความผิดพลาดของข้อมูล หรือความสะเพร่าไม่ตามมาตรฐาน
   - C ความชักช้า ไม่ทันใจ หรือคนทำงานไม่ตรงประเด็น
   - D ความขัดแย้งในทีม หรือการเปลี่ยนตารางเวรกะทันหัน

#### **หมวดการสื่อสาร**

7. เวลาสอนน้องพยาบาลจบใหม่…
   - A สร้างบรรยากาศสนุก เป็นกันเอง ให้น้องกล้าถาม
   - B อธิบายทฤษฎีและมาตรฐานการพยาบาลให้เข้าใจถ่องแท้
   - C เน้นให้ลองทำจริง แก้ไขจุดผิดพลาดตรงนั้นเลย
   - D สอนทีละขั้นตอนอย่างใจเย็น คอยประกบช่วยเหลือ
8. เมื่อคนไข้บ่นหรือโวยวาย…
   - A อธิบายกฎระเบียบและข้อเท็จจริงทางการแพทย์อย่างมีเหตุผล
   - B เข้าไปพูดคุย ประนีประนอม ให้คนไข้ใจเย็นลง
   - C เข้าไปจัดการปัญหาทันที อธิบายสั้นๆ ให้จบเรื่อง
   - D รับฟังอย่างตั้งใจและเห็นอกเห็นใจ (Empathy) ให้เขาได้ระบาย
9. ในที่ประชุมวอร์ด คุณมักจะ…
   - A เสนอไอเดียใหม่ๆ สร้างสีสันและชวนทุกคนคุย
   - B เป็นผู้ฟังที่ดี สนับสนุนความคิดเห็นทีมเพื่อความปรองดอง
   - C กล้าเสนอความเห็นที่แตกต่าง เพื่อให้งานเดินหน้า
   - D ตั้งคำถามเกี่ยวกับรายละเอียด หรือความเสี่ยงที่อาจเกิดขึ้น
10. คำชมที่คุณได้รับแล้วดีใจที่สุด…
   - A "คุณทำงานละเอียด รอบคอบ และมีความรู้แน่นมาก"
   - B "คุณเป็นคนเสมอต้นเสมอปลาย ไว้ใจได้ และพึ่งพาได้เสมอ"
   - C "คุณทำงานเร็ว ตัดสินใจเก่ง แก้ปัญหาได้เยี่ยม"
   - D "คุณเป็นคนน่ารัก อยู่ด้วยแล้วมีความสุข ทีมขาดคุณไม่ได้"
11. เมื่อทำงานร่วมกับแพทย์ที่อารมณ์ร้อน…
   - A สงบสติอารมณ์ ยอมๆ ไปก่อนเพื่อไม่ให้เกิดเรื่องใหญ่
   - B เตรียมข้อมูลคนไข้ให้พร้อมที่สุด เพื่อไม่ให้มีช่องว่างโดนตำหนิ
   - C ใช้มุกตลกหรือคำพูดหวานๆ ทำให้สถานการณ์ผ่อนคลาย
   - D กล้าโต้แย้งด้วยเหตุผลถ้าแพทย์ทำไม่ถูกต้อง
12. การสื่อสารในไลน์กลุ่มงาน…
   - A พิมพ์รายละเอียดครบถ้วน ตรวจทานคำผิดก่อนส่ง
   - B ตอบรับด้วยความสุภาพ (เช่น "รับทราบค่ะ", "ขอบคุณค่ะ")
   - C สั้น ห้วน ตรงประเด็น (เช่น "รับทราบ", "ด่วน")
   - D ใช้สติ๊กเกอร์เยอะๆ พิมพ์ยาวๆ มีอิโมจิ

#### **หมวดแรงจูงใจและการตัดสินใจ**

13. เป้าหมายในการทำงานพยาบาล…
   - A ความมั่นคง และการได้ช่วยเหลือผู้อื่นอย่างแท้จริง
   - B ความเป็นมืออาชีพ ความถูกต้องแม่นยำ และมาตรฐาน
   - C การมีสังคมที่ดี ได้รับการยอมรับจากเพื่อนร่วมงาน
   - D ความสำเร็จ ความก้าวหน้า และการเป็นที่ยอมรับในความสามารถ
14. เมื่อต้องตัดสินใจเรื่องสำคัญของคนไข้…
   - A รวบรวมข้อมูล วิเคราะห์ผลดีผลเสียอย่างละเอียดก่อน
   - B ปรึกษาหารือกับคนอื่นๆ เพื่อดูว่าใครคิดอย่างไร
   - C ตัดสินใจตามแนวทางที่เคยทำมา หรือถามรุ่นพี่เพื่อความอุ่นใจ
   - D ตัดสินใจทันทีตามสัญชาตญาณและประสบการณ์
15. คุณจะรู้สึกเครียดมากที่สุดเมื่อ…
   - A รู้สึกโดดเดี่ยว หรือเพื่อนร่วมงานไม่ชอบ
   - B ต้องทำงานแบบลวกๆ ไม่มีมาตรฐาน หรือข้อมูลไม่ชัดเจน
   - C รู้สึกว่าควบคุมสถานการณ์ไม่ได้ หรือถูกแย่งอำนาจ
   - D ต้องเจอการเปลี่ยนแปลงกะทันหัน หรือความขัดแย้งรุนแรง
16. หากวอร์ดนำระบบ IT ใหม่มาใช้…
   - A กังวลว่าจะใช้ยากไหม อยากมีเวลาฝึกฝนจนชัวร์
   - B อยากรู้รายละเอียดระบบ การสำรองข้อมูล และความถูกต้อง
   - C ตื่นเต้นที่จะได้ลองของใหม่ ชวนคนอื่นมาเล่นด้วย
   - D อยากรู้ว่าจะช่วยให้งานเสร็จเร็วขึ้นไหม ถ้าใช่ก็เอาเลย
17. สไตล์การให้คำแนะนำคนไข้ก่อนกลับบ้าน (Discharge Planning)…
   - A ให้ข้อมูลละเอียด ครบถ้วน อ้างอิงคู่มือ/มาตรฐานเป๊ะๆ
   - B อธิบายอย่างใจเย็น เป็นขั้นตอน (Step-by-step) คอยถามความเข้าใจ
   - C เน้นกระชับ ตรงประเด็น บอกเป้าหมายให้หายเร็วที่สุด
   - D เน้นพูดให้กำลังใจ สร้างความมั่นใจให้กลับไปใช้ชีวิตอย่างมีความสุข
18. เมื่อเพื่อนร่วมงานทำผิดพลาด…
   - A วิเคราะห์ว่าผิดเพราะอะไร หาทางป้องกันไม่ให้เกิดซ้ำ
   - B พูดตรงๆ ว่าผิดตรงไหน และบอกให้รีบแก้ไข
   - C ให้กำลังใจว่า "ไม่เป็นไรนะ เอาใหม่" รักษาน้ำใจ
   - D เข้าไปช่วยแก้ปัญหาเงียบๆ หรือช่วยแบ่งเบางาน

#### **หมวดพฤติกรรมภายใต้ความกดดัน**

19. เมื่อเวรยุ่งมากจนไม่ได้กินข้าว…
   - A เครียด กังวลว่าจะทำงานพลาด พยายามตรวจสอบซ้ำๆ
   - B ก้มหน้าก้มตาทำต่อไป อดทน ไม่บ่น เพื่อให้ทีมไม่เสียขวัญ
   - C บ่นระบายกับเพื่อน หาเรื่องตลกมาคุยคลายเครียด
   - D หงุดหงิด สั่งการเสียงดัง เร่งรีบเพื่อให้งานจบ
20. เมื่อมีความเห็นไม่ตรงกันในทีม…
   - A ยกกฎระเบียบหรือหลักฐานมายืนยันความถูกต้อง
   - B ยอมตามเสียงส่วนใหญ่ เพื่อรักษาความสงบสุข
   - C พยายามไกล่เกลี่ย หรือเปลี่ยนเรื่องคุยลดความตึงเครียด
   - D ยืนกรานในความคิดตนเอง พยายามเอาชนะด้วยเหตุผล
21. สิ่งที่กระตุ้นให้ทำงานได้ดีขึ้น…
   - A คำชมเชย รางวัล หรือการทำกิจกรรมร่วมกับผู้อื่น
   - B ความท้าทาย หรือโจทย์ยากๆ ที่ต้องพิสูจน์ฝีมือ
   - C การมีเวลาทำงานละเอียด และข้อมูลที่ครบถ้วน
   - D ความมั่นคง ปลอดภัย และขั้นตอนทำงานที่ชัดเจน
22. เพื่อนร่วมงานมองว่าคุณเป็นคน…
   - A ใจดี พึ่งพาได้ แต่อาจจะเชื่องช้าหรือไม่กล้าตัดสินใจ
   - B เป๊ะ เจ้าระเบียบ รอบคอบ แต่อาจจะจู้จี้จุกจิก
   - C คุยเก่ง มนุษยสัมพันธ์ดี แต่อาจจะไม่ค่อยรอบคอบ
   - D เก่ง กล้าตัดสินใจ แต่อาจจะดุหรือใจร้อน
23. เมื่อต้องวางแผนตารางเวร…
   - A ต้องยุติธรรม นับชั่วโมงการทำงานให้เท่ากันเป๊ะๆ
   - B แบบไหนก็ได้ที่ทุกคนโอเค ไม่อยากให้ใครลำบากใจ
   - C อยากได้เวรที่ได้อยู่กับเพื่อนสนิท หรือได้หยุดไปเที่ยว
   - D อยากให้จัดเสร็จเร็วๆ เอาตามความเหมาะสมของงานเป็นหลัก
24. หากมีนโยบายใหม่ที่คุณไม่เห็นด้วย…
   - A เขียนบันทึกข้อความชี้แจงข้อดีข้อเสียเป็นลายลักษณ์อักษร
   - B ทำตามนโยบายไปเงียบๆ แม้จะไม่เห็นด้วยก็ตาม
   - C จับกลุ่มคุยกับเพื่อนๆ เพื่อแสดงความคิดเห็น (บ่น)
   - D ลุกขึ้นถามผู้บริหารทันทีว่าทำไมต้องทำแบบนี้

## วิธีแปลผล (Interpretation Rules)

กฎการแปลผล: คุณคือคนแบบไหนในวอร์ด?  
เมื่อรวมคะแนนแล้ว ให้ดูว่าตัวอักษรใดมีคะแนนสูงสุด แล้วดูคำทำนายด้านล่างนี้:

### 🐂 ถ้าคุณได้ A มากที่สุด: คุณคือ "กระทิง" (The Bull) - D Style

**(ผู้นำจอมพลัง ผู้มุ่งมั่นพิชิตเป้าหมาย / Dominance)**

- **นิยาม:** เปรียบเสมือน **กระทิง** ที่มีพละกำลังมหาศาล พร้อมพุ่งชนเป้าหมาย ชอบความรวดเร็ว และไม่เกรงกลัวอุปสรรค
- **จุดแข็งในงานพยาบาล:** ตัดสินใจเด็ดขาดในภาวะวิกฤต (เช่น CPR) ทำงานแข่งกับเวลาได้ดีมาก แก้ปัญหาเฉพาะหน้าเก่ง เน้นผลลัพธ์ว่า "คนไข้ต้องรอด" หรือ "งานต้องเสร็จ"
- **ข้อควรระวัง:** อาจเผลอใช้อารมณ์หรือคำพูดที่ดุดัน สั้นห้วน จนเพื่อนร่วมงานรู้สึกกลัว หรือลืมใส่ใจรายละเอียดเล็กน้อยเพราะมุ่งแต่เป้าหมายหลัก
- **บทบาทที่เหมาะสม:** หัวหน้าเวร, พยาบาล ER/ICU, หรือทีม Code Blue ที่ต้องการความเฉียบขาด

### 🦅 ถ้าคุณได้ B มากที่สุด: คุณคือ "อินทรี" (The Eagle) - I Style

**(นักวิสัยทัศน์ ผู้สร้างแรงบันดาลใจ / Influence)**

- **นิยาม:** เปรียบเสมือน **นกอินทรี** ที่บินสูง มองเห็นภาพรวมกว้างไกล รักอิสระ ชอบเรียนรู้สิ่งใหม่ๆ และมีทักษะการสื่อสารที่ยอดเยี่ยม
- **จุดแข็งในงานพยาบาล:** สร้างบรรยากาศเชิงบวกในวอร์ดได้เก่ง ประสานงานกับสหสาขาวิชาชีพได้ราบรื่น มีวาทศิลป์ในการพูดคุยให้กำลังใจคนไข้และญาติ มองเห็นโอกาสในการพัฒนางานใหม่ๆ
- **ข้อควรระวัง:** อาจเบื่อง่ายกับงานเอกสารซ้ำๆ (Routine) หรือมัวแต่คุยโปรเจกต์ใหม่จนลืมลงรายละเอียดในเนื้องานปัจจุบัน
- **บทบาทที่เหมาะสม:** งานประชาสัมพันธ์, ทีมรับรองคุณภาพ (HA) ด้านการสื่อสาร, หรือหน่วยงานที่ต้องดีลกับคนเยอะๆ

### 🐭 ถ้าคุณได้ C มากที่สุด: คุณคือ "หนู" (The Mouse) - S Style

**(ผู้ประสานใจ ผู้ดูแลด้วยความห่วงใย / Steadiness)**

- **นิยาม:** เปรียบเสมือน **หนู** ตัวเล็กที่ว่องไวแต่อ่อนโยน ช่างสังเกตความรู้สึก ชอบอยู่รวมกันเป็นกลุ่ม รักความสงบ และดูแลใส่ใจคนรอบข้างเสมอ
- **จุดแข็งในงานพยาบาล:** มีความเห็นอกเห็นใจ (Empathy) สูงมาก เป็นผู้ฟังที่ดีเยี่ยม รับรู้อารมณ์คนไข้ได้ไว ให้การพยาบาลที่นุ่มนวลและใส่ใจรายละเอียดทางจิตใจ เพื่อนร่วมงานรักเพราะช่วยเหลือเก่ง
- **ข้อควรระวัง:** ขี้เกรงใจ ไม่กล้าปฏิเสธจนงานล้นมือ หรือเก็บความเครียดไว้คนเดียว ไม่ชอบความขัดแย้งและการเปลี่ยนแปลงกะทันหัน
- **บทบาทที่เหมาะสม:** พยาบาลพี่เลี้ยง (Preceptor), ดูแลผู้ป่วยเรื้อรัง/ประคับประคอง (Palliative), หรือจิตเวช

### 🧸 ถ้าคุณได้ D มากที่สุด: คุณคือ "หมี" (The Bear) - C Style

**(นักวิเคราะห์ ผู้รักษากฎระเบียบ / Conscientiousness)**

- **นิยาม:** เปรียบเสมือน **หมี** ที่มั่นคง หนักแน่น ใช้เวลาในการไตร่ตรอง วางแผนอย่างรอบคอบ ยึดมั่นในหลักการ ทฤษฎี และความปลอดภัยสูงสุด
- **จุดแข็งในงานพยาบาล:** ทำงานเป็นระบบระเบียบมาก ข้อมูลแน่น (Evidence-based) แม่นยำเรื่องยาและคำนวณ ความผิดพลาดน้อยมาก ยึดถือมาตรฐานวิชาชีพอย่างเคร่งครัด
- **ข้อควรระวัง:** อาจใช้เวลาตัดสินใจนานเพราะต้องการข้อมูลที่สมบูรณ์แบบ (Perfectionist) หรือดูเคร่งขรึม เข้าถึงยากในสายตาน้องๆ อาจหงุดหงิดง่ายถ้าเห็นอะไรไม่เป็นระเบียบ
- **บทบาทที่เหมาะสม:** งานควบคุมการติดเชื้อ (IC), งานประกันคุณภาพ, ห้องผ่าตัด (OR), หรือเภสัชกรรม

## การนำผลลัพธ์ DISC ไปใช้พัฒนาทีมงานพยาบาลในวอร์ด

การนำผลลัพธ์จากแบบประเมิน DISC (ในบริบทสัตว์ 4 ทิศ: กระทิง, อินทรี, หนู, หมี) ไปใช้พัฒนาทีมงานพยาบาลในวอร์ด สามารถทำได้โดยเน้นการ **"ใช้คนให้ถูกงาน"** และ **"สื่อสารให้ถูกจริต"** ซึ่งจะช่วยลดความขัดแย้งและเพิ่มประสิทธิภาพการดูแลผู้ป่วยได้จริง

### 1) การจัดมอบหมายงานให้ตรงกับจุดแข็ง

การรู้ว่าพยาบาลแต่ละคนเป็น “สัตว์” ชนิดไหน ช่วยให้หัวหน้าหอผู้ป่วยมอบหมายงานพิเศษหรือบทบาทเฉพาะได้เหมาะสมยิ่งขึ้น:

- **🐂 กลุ่มกระทิง (D - Dominance):**
  - **งานที่เหมาะสม:** มอบหมายให้เป็น **ผู้นำทีม (Team Leader)** ในเวรดึก หรือสถานการณ์วิกฤต (เช่น Code Blue) เพราะกล้าตัดสินใจและชอบความท้าทาย ให้ดูแลงานที่ต้องการผลลัพธ์รวดเร็ว เช่น การบริหารจัดการเตียง หรือเคลียร์ปัญหาข้อร้องเรียนที่ต้องการความเด็ดขาด
  - **การพัฒนา:** ท้าทายด้วยเป้าหมายที่ยากขึ้น แต่ต้องคอยเตือนเรื่องความละเอียดอ่อนทางอารมณ์กับเพื่อนร่วมงาน

- **🦅 กลุ่มอินทรี (I - Influence):**
  - **งานที่เหมาะสม:** มอบหมายงาน **ประชาสัมพันธ์, กิจกรรมสันทนาการในวอร์ด** หรือเป็นผู้นำในการทำกลุ่มบำบัด/สอนสุขศึกษาให้คนไข้ เพราะมีทักษะการพูดโน้มน้าวและสร้างบรรยากาศเชิงบวก
  - **การพัฒนา:** ช่วยฝึกเรื่องการลงรายละเอียดและการติดตามงานให้จบ เพราะกลุ่มนี้อาจเริ่มโปรเจกต์เก่งแต่ปิดจบยาก

- **🐭 กลุ่มหนู (S - Steadiness):**
  - **งานที่เหมาะสม:** เหมาะกับการเป็น **พี่เลี้ยง (Preceptor)** ให้น้องใหม่ หรือดูแลผู้ป่วย Palliative Care ที่ต้องการความใส่ใจสูง เพราะอดทน เป็นผู้ฟังที่ดี และสร้างความอุ่นใจ งานเอกสารหรืองาน Routine ที่ต้องการความสม่ำเสมอทำได้ดี
  - **การพัฒนา:** สนับสนุนให้กล้าแสดงความคิดเห็นมากขึ้น และอย่าปล่อยให้แบกภาระงานเกินตัว (Burnout) เพราะกลุ่มนี้มักปฏิเสธคนไม่เป็น

- **🧸 กลุ่มหมี (C - Compliance/Conscientiousness):**
  - **งานที่เหมาะสม:** มอบหมายงาน **ประกันคุณภาพ (HA/QA), งานควบคุมการติดเชื้อ (IC)** หรือการตรวจสอบความถูกต้องของคำสั่งการรักษาและยา เพราะละเอียด รอบคอบ และแม่นยำในมาตรฐาน
  - **การพัฒนา:** ช่วยลดความเครียดจากความสมบูรณ์แบบ (Perfectionism) และฝึกให้ยืดหยุ่นในสถานการณ์ที่ไม่วิกฤต

### 2) การปรับวิธีการสื่อสารในทีม

ปัญหาส่วนใหญ่ในวอร์ดเกิดจากการสื่อสารผิดรูปแบบ การเข้าใจ DISC จะช่วยสร้าง **"ภาษากลาง"** ในการคุยกัน:

- **คุยกับกระทิง (D):** **"สั้น กระชับ ตรงประเด็น"**
  - ไม่ต้องเกริ่นนำนาน บอกเลยว่าปัญหาคืออะไร ต้องการให้ทำอะไร และผลลัพธ์คืออะไร (แบบ SBAR)
- **คุยกับอินทรี (I):** **"เป็นกันเอง ให้โอกาสพูด เปิดช่องให้แสดงไอเดีย"**
  - เริ่มด้วยเรื่องสัพเพเหระเล็กน้อย สร้างบรรยากาศผ่อนคลาย แล้วค่อยเข้าเรื่องงาน อย่าเน้นรายละเอียดเอกสารมากเกินไป
- **คุยกับหนู (S):** **"นุ่มนวล เป็นขั้นตอน ให้ความมั่นใจ"**
  - พูดด้วยน้ำเสียงอบอุ่น ถามสารทุกข์สุกดิบ อธิบายขั้นตอนการเปลี่ยนแปลงช้าๆ อย่ากดดันหรือเร่งรัด
- **คุยกับหมี (C):** **"เตรียมข้อมูลพร้อม อ้างอิงหลักการ"**
  - คุยด้วยข้อเท็จจริง มีหลักฐานอ้างอิง อย่าใช้อารมณ์ และให้เวลาเขาได้คิดวิเคราะห์ก่อนตอบ

### 3) การบริหารความขัดแย้ง (Conflict Resolution)

เมื่อเกิดความขัดแย้งในทีม หัวหน้าสามารถใช้ DISC วิเคราะห์สาเหตุได้:

- **คู่กระทิง vs หมี:** มักตีกันเรื่อง "ความเร็ว" vs "ความละเอียด" (กระทิงจะเอางานเสร็จเดี๋ยวนี้ แต่หมีจะเอาเป๊ะ) → _ทางแก้:_ ให้ตกลง Deadline และมาตรฐานขั้นต่ำที่ยอมรับได้ร่วมกัน
- **คู่กระทิง vs กระทิง:** แย่งกันนำ แย่งกันพูด → _ทางแก้:_ แบ่งขอบเขตอำนาจหน้าที่ (Zone) ให้ชัดเจนว่าใครตัดสินใจเรื่องไหน

### 4) การสร้างสมดุลทีม

ใช้ผลลัพธ์มาดูภาพรวมของวอร์ดว่าเรามีสัตว์ชนิดไหนมากหรือน้อยเกินไป:

- **ถ้ามี "หนู" เยอะเกินไป:** วอร์ดจะสงบสุข รักใคร่กลมเกลียว แต่การเปลี่ยนแปลงจะเกิดยาก และการตัดสินใจในภาวะวิกฤตอาจล่าช้า
- **ถ้ามี "กระทิง" เยอะเกินไป:** งานเดินเร็วมาก แต่บรรยากาศตึงเครียด และอาจมีการโต้เถียงกันบ่อย
- **ทางแก้:** ในการจัดเวรพยายามเกลี่ยให้แต่ละเวรมีสัตว์คละกัน เช่น มีกระทิงนำทีม มีหนูคอยประสาน และมีหมีคอยตรวจสอบยา เพื่อให้การดูแลคนไข้ครอบคลุมทุกมิติ

### สรุป

การใช้ DISC ในวอร์ดไม่ใช่การตีตราว่าใครเป็นคนอย่างไร แต่เพื่อให้เข้าใจ **"แรงจูงใจ"** และ **"ความกลัว"** ของน้องในทีม เพื่อให้หัวหน้าสามารถดูแลใจและใช้งานพวกเขาได้อย่างมีประสิทธิภาพสูงสุด

## คำอธิบาย 4 มิติ + แนวทางใช้ในงานพยาบาล/การให้คำปรึกษา

นิยามสั้น ๆ:

- D มักตรง เร็ว เน้นผลลัพธ์
- i มักเปิดเผย ชวนคุย สร้างพลัง
- S มักใจเย็น อดทน เน้นความร่วมมือ
- C มักวิเคราะห์ รอบคอบ เน้นมาตรฐาน/ความถูกต้อง

### D (Dominance) สูง

เด่น: ตัดสินใจเร็ว กล้ารับผิดชอบ เหมาะกับสถานการณ์ฉุกเฉิน/งานที่ต้อง “ปิดงาน”

เสี่ยง: สื่อสารตรงเกินไป กดดันคนอื่น เผลอมองข้ามอารมณ์คน

ทริค: ระวัง “รีบแก้ปัญหาแทน” ใช้คำถามปลายเปิดก่อนให้คำแนะนำ

### i (Influence) สูง

เด่น: สร้างสัมพันธภาพเก่ง ดึงความร่วมมือ เหมาะกับงานสื่อสาร/การเปลี่ยนแปลง

เสี่ยง: เผลอข้ามรายละเอียด ตัดสินใจจากความรู้สึก/บรรยากาศ

ทริค: ใช้โครง “สรุป-ตกลงแผน-ติดตาม” เพื่อไม่หลุดประเด็น

### S (Steadiness) สูง

เด่น: ใจเย็น สม่ำเสมอ ดูแลคนรอบข้าง เหมาะกับงานต่อเนื่อง/ดูแลระยะยาว

เสี่ยง: หลีกเลี่ยงความขัดแย้ง ไม่กล้าปฏิเสธ ช้าในการเปลี่ยนแปลง

ทริค: ฝึก “ขอบเขต (boundary)” และประโยคยืนยันสิทธิอย่างสุภาพ

### C (Conscientiousness) สูง

เด่น: รอบคอบ แม่นมาตรฐาน ลดความเสี่ยง เหมาะกับงานคุณภาพ/ความปลอดภัย

เสี่ยง: คิดนาน เพอร์เฟกชันนิสม์ วิจารณ์ตัวเอง/คนอื่นสูง

ทริค: เพิ่มการสะท้อนความรู้สึก (ไม่ยึดแต่ข้อมูล) และกำหนด “พอแล้วที่ดีพอ”

## การแปลผลแบบคู่สไตล์ (Top 2)

- D/i: มุ่งผล + โน้มน้าวเก่ง (ขับเคลื่อนทีมเร็ว) ระวังพูดเร็ว/แรง
- i/D: เข้าหาคน + กล้าชน (นำแบบมีเสน่ห์) ระวังใจร้อน/ตัดสินเร็ว
- D/C: เด็ดขาด + มาตรฐานสูง (ยืนบนเหตุผล) ระวังเคร่งและคาดหวังสูง
- C/D: วิเคราะห์ + กล้าตัดสินเมื่อมั่นใจ (คมและชัด) ระวังช้าในช่วงข้อมูลไม่ครบ
- D/S: เร็วแต่ยังคุมความนิ่งได้ (จัดระบบคน) ระวังสลับโหมดเร็ว-ช้าให้ทีมงง
- S/D: ใจเย็นแต่ยืนหลักได้ (ประคองแล้วค่อยตัดสิน) ระวังเก็บอารมณ์จนระเบิด
- i/S: เป็นมิตร อบอุ่น ประสานทีมเก่ง (เหมาะงานคน) ระวังเลี่ยงความขัดแย้ง
- S/i: สุขุมแต่เข้าถึงง่าย (ดูแลคนแบบนุ่ม) ระวังไม่กล้าพูดสิ่งยาก
- i/C: สื่อสารดี + มีเหตุผล (พรีเซนต์ข้อมูลเก่ง) ระวังเครียดเมื่อรายละเอียดไม่พร้อม
- C/i: ข้อมูลแน่น + คุยรู้เรื่อง (อธิบายเก่ง) ระวังวิจารณ์มากไปในเวลาจำกัด
- S/C: มั่นคง + รอบคอบ (งานคุณภาพสม่ำเสมอ) ระวังช้าเมื่อมีแรงกดดัน
- C/S: ระเบียบ + ใจเย็น (มาตรฐานและความปลอดภัย) ระวังต้านการเปลี่ยนแปลง
