แบบสอบถามฉบับสั้น (24 ข้อ)

คำชี้แจงผู้ทำแบบประเมิน

เลือกคำตอบเพียง 1 ข้อในแต่ละคำถามที่ “ใกล้เคียงคุณที่สุด” ในสถานการณ์ทั่วไป (ไม่ใช่วันที่เครียดสุดหรือดีสุด) ไม่มีถูกหรือผิด

(สำหรับพิมพ์/อ้างอิง)
  1. เริ่มเวรแล้วงานค้างเยอะ คุณมักจะ…
    • A ตรวจสอบรายละเอียดงานค้างทั้งหมดก่อนเพื่อวางแผนให้แม่นยำ
    • B ชวนคุยทีมให้ฮึกเหิม แล้วเริ่มลงมือแบบ “ไปด้วยกัน”
    • C กำหนดเป้าหมายและจัดลำดับงานเร่งด่วนทันที
    • D เข้าช่วยเหลือและสนับสนุนเพื่อนเพื่อให้งานลื่นไหล ลดความตึงเครียด
  2. เมื่อเกิดเหตุฉุกเฉิน (CPR) บทบาทที่คุณถนัดคือ…
    • A ผู้จดบันทึกและตรวจสอบขนาดยา/เวลา (Recorder/Timekeeper) อย่างแม่นยำ
    • B ผู้ช่วยคอยส่งเครื่องมือและดูแลความเรียบร้อย
    • C ผู้นำสั่งการ (Team Leader) ตัดสินใจฉับไว
    • D ผู้ประสานงาน สื่อสารกับญาติและทีมด้วยความกระตือรือร้น
  3. เมื่อต้องเรียนรู้อุปกรณ์การแพทย์ใหม่…
    • A รอให้คนอื่นใช้จนชินก่อน แล้วค่อยเรียนรู้ตามขั้นตอนอย่างช้าๆ
    • B อ่านคู่มืออย่างละเอียด ศึกษาข้อมูลเทคนิคก่อนเริ่มใช้
    • C ลองจับลองทำเลย อยากรู้การทำงานเพื่อให้ใช้ได้เร็วที่สุด
    • D ชอบเรียนรู้ไปพร้อมๆ กับเพื่อนๆ ในวอร์ด ให้คนอื่นช่วยสอน
  4. เมื่อแพทย์เขียนคำสั่งการรักษาไม่ชัดเจน…
    • A ถามเพื่อนร่วมงานว่า "หมอคนนี้ปกติเขียนแบบนี้หมายความว่าไง?"
    • B ตรวจสอบประวัติเก่าและแนวทางก่อน แล้วค่อยถามเพื่อความชัวร์
    • C โทรหาแพทย์ทันทีเพื่อถามให้รู้เรื่อง กันความล่าช้า
    • D พยายามแกะลายมือเองก่อน หรือรอจังหวะหมออารมณ์ดีค่อยถาม
  5. สไตล์การส่งเวรของคุณ…
    • A เล่าเรื่องราว แลกเปลี่ยนความเห็นกับเพื่อนร่วมเวร
    • B รวดเร็ว กระชับ เน้นประเด็นสำคัญและแผนที่ต้องทำต่อ
    • C เป็นระเบียบ เรียงลำดับข้อมูลครบถ้วน อ้างอิงผล Lab เป๊ะๆ
    • D พูดด้วยน้ำเสียงนุ่มนวล ใส่ใจรายละเอียด
  6. สิ่งที่คุณไม่ชอบที่สุดในเวลางาน…
    • A บรรยากาศเงียบเหงา หรือถูกเมิน
    • B ความผิดพลาด หรือความสะเพร่าทั้งของตนเองและผู้อื่น
    • C ความชักช้า ไม่ทันใจ
    • D ความขัดแย้งในทีม หรือการเปลี่ยนกะทันหัน
  7. เวลาสอนน้องพยาบาลจบใหม่…
    • A สร้างบรรยากาศสนุก เป็นกันเอง ให้น้องกล้าถาม
    • B อธิบายทฤษฎีและมาตรฐานการพยาบาลให้เข้าใจถ่องแท้
    • C เน้นให้ลองทำจริง แก้ไขจุดผิดพลาดตรงนั้นเลย
    • D สอนทีละขั้นตอนอย่างใจเย็น คอยประกบช่วยเหลือ
  8. เมื่อคนไข้บ่นหรือโวยวาย…
    • A อธิบายกฎระเบียบและข้อเท็จจริงทางการแพทย์อย่างมีเหตุผล
    • B เข้าไปพูดคุย ประนีประนอม ให้คนไข้ใจเย็นลง
    • C เข้าไปจัดการปัญหาทันที อธิบายสั้นๆ ให้จบเรื่อง
    • D รับฟังอย่างตั้งใจและเห็นอกเห็นใจให้คนไข้ได้ระบาย
  9. ในที่ประชุมวอร์ด คุณมักจะ…
    • A เสนอไอเดียใหม่ๆ สร้างสีสันและชวนทุกคนคุย
    • B เป็นผู้ฟังที่ดี สนับสนุนความคิดเห็นทีม
    • C เสนอความเห็นที่แตกต่าง เพื่อให้งานเดินหน้า
    • D ตั้งคำถามเกี่ยวกับรายละเอียด หรือความเสี่ยงที่อาจเกิดขึ้น
  10. คำชมที่คุณได้รับแล้วดีใจที่สุด…
    • A "คุณทำงานละเอียด รอบคอบ และมีความรู้แน่นมาก"
    • B "คุณเป็นคนเสมอต้นเสมอปลาย ไว้ใจได้ และพึ่งพาได้เสมอ"
    • C "คุณทำงานเร็ว ตัดสินใจเก่ง แก้ปัญหาได้เยี่ยม"
    • D "คุณเป็นคนน่ารัก อยู่ด้วยแล้วมีความสุข ทีมขาดคุณไม่ได้"
  11. เมื่อทำงานร่วมกับแพทย์ที่อารมณ์ร้อน…
    • A สงบสติอารมณ์ ยอมๆ ไปก่อนเพื่อไม่ให้เกิดเรื่องใหญ่
    • B เตรียมข้อมูลคนไข้ให้พร้อมที่สุด เพื่อไม่ให้มีช่องว่างโดนตำหนิ
    • C ใช้มุกตลกหรือคำพูดหวานๆ ทำให้สถานการณ์ผ่อนคลาย
    • D กล้าโต้แย้งด้วยเหตุผลถ้าแพทย์ทำไม่ถูกต้อง
  12. การสื่อสารในไลน์กลุ่มงาน…
    • A พิมพ์รายละเอียดครบถ้วน ตรวจทานคำผิดก่อนส่ง
    • B ตอบรับด้วยความสุภาพ (เช่น "รับทราบค่ะ", "ขอบคุณค่ะ")
    • C สั้น ห้วน ตรงประเด็น (เช่น "รับทราบ", "ด่วน")
    • D ใช้สติ๊กเกอร์เยอะๆ พิมพ์ยาว
  13. เป้าหมายในการทำงานพยาบาล…
    • A ความมั่นคง
    • B ความความก้าวหน้า
    • C การได้รับการยอมรับ
    • D ความสำเร็จ
  14. เมื่อต้องตัดสินใจเรื่องสำคัญของคนไข้…
    • A รวบรวมข้อมูล วิเคราะห์ผลดีผลเสียอย่างละเอียดก่อน
    • B ปรึกษาหารือกับคนอื่นๆ เพื่อดูว่าใครคิดอย่างไร
    • C ตัดสินใจตามแนวทางที่เคยทำมา หรือถามรุ่นพี่เพื่อความอุ่นใจ
    • D ตัดสินใจทันทีตามสัญชาตญาณและประสบการณ์
  15. คุณจะรู้สึกเครียดมากที่สุดเมื่อ…
    • A รู้สึกโดดเดี่ยว หรือเพื่อนร่วมงานไม่ชอบ
    • B ได้รับข้อมูลไม่ชัดเจน หรือไม่ได้รับเวร
    • C รู้สึกว่าควบคุมสถานการณ์ไม่ได้ หรือรู้สึกว่าถูกบังคับ
    • D ต้องเจอการเปลี่ยนแปลงกะทันหัน หรือความขัดแย้งรุนแรง
  16. หากวอร์ดนำระบบใหม่มาใช้…
    • A กังวลว่าจะใช้ยากไหม อยากมีเวลาฝึกฝนจนชัวร์
    • B อยากรู้รายละเอียดระบบ การสำรองข้อมูล และความถูกต้อง
    • C ตื่นเต้นที่จะได้ลองของใหม่ ชวนคนอื่นมาเล่นด้วย
    • D อยากรู้ว่าจะช่วยให้งานเสร็จเร็วขึ้นไหม ถ้าใช่ก็เอาเลย
  17. สไตล์การให้คำแนะนำคนไข้ก่อนกลับบ้าน…
    • A ให้ข้อมูลละเอียด ครบถ้วน อ้างอิงคู่มือ/มาตรฐานเป๊ะๆ
    • B อธิบายอย่างใจเย็น เป็นขั้นตอนคอยถามความเข้าใจ
    • C เน้นกระชับ ข้อเท็จจริง ตรงประเด็น เน้นเนื้้อ ไม่เน้นน้ำ
    • D เน้นพูดให้กำลังใจ สร้างความมั่นใจให้กลับไปใช้ชีวิตอย่างมีความสุข
  18. เมื่อเพื่อนร่วมงานทำผิดพลาด…
    • A วิเคราะห์ว่าผิดเพราะอะไร หาทางป้องกันไม่ให้เกิดซ้ำ
    • B พูดตรงๆ ว่าผิดตรงไหน และบอกให้รีบแก้ไข
    • C ให้กำลังใจว่า "ไม่เป็นไรนะ เอาใหม่" รักษาน้ำใจ
    • D เข้าไปช่วยแก้ปัญหาเงียบๆ หรือช่วยแบ่งเบางาน
  19. เมื่อเวรยุ่งมากจนไม่ได้กินข้าว…
    • A เครียด กังวลว่าจะทำงานพลาด
    • B ก้มหน้าก้มตาทำต่อไป อดทน ไม่บ่น เพื่อให้ทีมไม่เสียขวัญ
    • C บ่นระบายกับเพื่อน หาเรื่องตลกมาคุยคลายเครียด
    • D หงุดหงิด เร่งรีบเพื่อให้งานจบ
  20. เมื่อมีความเห็นไม่ตรงกันในทีม…
    • A ยกกฎระเบียบหรือหลักฐานมายืนยันความถูกต้อง
    • B ยอมตามเสียงส่วนใหญ่ เพื่อรักษาความสงบสุข
    • C พยายามไกล่เกลี่ย หรือเปลี่ยนเรื่องคุยลดความตึงเครียด
    • D ยืนกรานในความคิดตนเอง พยายามเอาชนะด้วยเหตุผล
  21. สิ่งที่กระตุ้นให้ทำงานได้ดีขึ้น…
    • A คำชมเชย รางวัล หรือการทำกิจกรรมร่วมกับผู้อื่น
    • B ความท้าทาย หรือโจทย์ยากๆ ที่ต้องพิสูจน์ฝีมือ
    • C การมีเวลาทำงานละเอียด และข้อมูลที่ครบถ้วน
    • D ความมั่นคง ปลอดภัย และขั้นตอนทำงานที่ชัดเจน
  22. เพื่อนร่วมงานมองว่าคุณเป็นคน…
    • A ใจดี พึ่งพาได้
    • B ละเอียด รอบคอบ
    • C คุยเก่ง มนุษยสัมพันธ์ดี
    • D เก่ง กล้าตัดสินใจ
  23. เมื่อต้องวางแผนตารางเวร…
    • A ต้องยุติธรรม นับชั่วโมงการทำงานให้เท่ากันเป๊ะๆ
    • B แบบไหนก็ได้ที่ทุกคนโอเค ไม่อยากให้ใครลำบากใจ
    • C อยากได้เวรที่ได้อยู่กับเพื่อนสนิท หรือได้หยุดไปเที่ยว
    • D อยากให้จัดเสร็จเร็วๆ เอาตามความเหมาะสมของงานเป็นหลัก
  24. หากมีนโยบายใหม่ที่คุณไม่เห็นด้วย…
    • A เขียนบันทึกข้อความชี้แจงข้อดีข้อเสียเป็นลายลักษณ์อักษร
    • B ทำตามนโยบายไปเงียบๆ แม้จะไม่เห็นด้วยก็ตาม
    • C จับกลุ่มคุยกับเพื่อนๆ เพื่อแสดงความคิดเห็น (บ่น)
    • D ลุกขึ้นถามผู้บริหารทันทีว่าทำไมต้องทำแบบนี้

การอ่านผล Big Five (OCEAN) — ตนเอง ผู้อื่น และบริบทวอร์ด

ส่วนนี้สรุปจากงานวิจัยด้านบุคลิกภาพและการทำงานเป็นทีมในองค์กรและสายพยาบาล ใช้เป็น แนวทางสะท้อนตนเอง ไม่ใช่การวินิจฉัย ไม่ใช่เกณฑ์คัดเลือก และไม่ได้บอกว่าใคร “เก่ง” หรือ “อ่อน” แค่ชี้ แนวโน้ม ที่มักปรากฏในสถานการณ์ทั่วไป

ความหมายของแต่ละมิติ (ภาพรวมสั้น ๆ)

  • เปิดรับประสบการณ์ (O) — ความอยากรู้อยากเห็น จินตนาการ และความสบายใจกับสิ่งใหม่ ๆ / ความคลุมเครือ เทียบกับความชอบกิจวัตรและสิ่งที่คุ้นเคย
  • ความมีวินัย (C) — การจัดระเบียบ ความน่าเชื่อถือ การทำตามแผน และความละเอียดรอบคอบ เทียบกับความยืดหยุ่นและความเป็นอิสระในรูปแบบงาน
  • การเข้าหาสังคม (E) — พลังทางสังคม การแสดงออก และความต้องการปฏิสัมพันธ์ เทียบกับการฟื้นฟูพลังในความเงียบและการคิดก่อนพูด
  • ความเห็นแก่ผู้อื่น (A) — ความร่วมมือ ความเห็นอกเห็นใจ และการรักษาความสัมพันธ์ เทียบกับความตรงไปตรงมา การแข่งขัน หรือการตั้งคำถามเชิงวิพากษ์
  • ความไวต่อความเครียด (N) — ความไวต่อสิ่งเร้า อารมณ์แปรปรวน หรือความกังวล เทียบกับความนิ่งและความมั่นคงทางอารมณ์ภายใต้แรงกดดัน (คะแนนสูง = ไวต่อความเครียดมากขึ้น ไม่ได้แปลว่า “อ่อนแอ”)

ในงานวิจัยหลายชุด ความมีวินัย (C) เป็นตัวทำนายผลการทำงานที่แข็งแรงในหลายอาชีพ ส่วนมิติอื่นมีบทบาทกับ ความเข้ากันได้ในทีม การจัดการความเครียด และสไตล์การสื่อสาร มากกว่าการบอก “ทำงานเก่งหรือไม่” เพียงอย่างเดียว

กับตนเอง — ใช้อย่างไรให้เป็นประโยชน์

  • มองว่าเป็นแนวโน้ม ไม่ใช่คำตัดสิน: พฤติกรรมและทักษะ (เช่น การสื่อสาร การจัดลำดับความสำคัญ การควบคุมอารมณ์) ฝึกได้แม้จะไม่ตรงกับ “สไตล์ดั้งเดิม”
  • เน้นจุดแข็ง แต่รู้จุดที่มักต้องเสริม: คะแนน C สูงช่วยเรื่องความน่าไว้วางใจ แต่ระวังภาระเกินและ perfectionism; O สูงช่วยปรับตัวเมื่อมีนวัตกรรม แต่ระวังงานซ้ำ ๆ ที่จำเป็นต่อความปลอดภัย; A สูงช่วยทีม แต่ต้องฝึกตั้งขอบเขตและพูดเรื่องยากเมื่อจำเป็น
  • การเข้าหาสังคม (E) ต่ำไม่ใช่ข้อบกพร่อง: ให้เวลาพักฟื้น ใช้ข้อความหรือการสื่อสารแบบมีเวลาคิดเมื่อเหมาะสม
  • เมื่อไวต่อความเครียด (N) สูง: งานวิจัยเชื่อมโยงแนวโน้มนี้กับการใช้ทรัพยากรจัดการอารมณ์และการควบคุมตนมากขึ้น — จึงควรเน้น โครงสร้างที่ชัด (ตาราง มอบหมาย การนอน) การระบายกับคนที่ปลอดภัย การ debrief หลังเหตุการณ์หนัก และการขอความช่วยเหลือจากระบบสนับสนุน ไม่ใช่แค่ “ฮึดสู้คนเดียว”

กับผู้อื่นและที่ทำงาน — ปรับการสื่อสารให้ตรงแนวโน้ม (ไม่ใช่ติดป้ายคน)

หลักคือ อย่าพยายามเปลี่ยนบุคลิกของอีกฝ่าย แต่ปรับวิธีคุย ข้อมูลที่เตรียม และจังหวะให้งานเดินต่อได้อย่างปลอดภัย

แนวโน้มของอีกฝ่าย มักช่วยได้ถ้าเรา…
O สูงอธิบายเหตุผลของการเปลี่ยนแปลง เปิดพื้นะให้ลองแนวทางใหม่เมื่อปลอดภัย
C สูงให้กำหนดเวลา เกณฑ์คุณภาพ และข้อมูลที่ตรวจสอบได้ ชัดเจน
E สูงใช้การพูดคุยสั้น ๆ ประชุมเปิด หรือชวนระดมความคิด — แต่แยกจากคนที่ต้องการเวลาคิดเงียบ ๆ
A สูงเปิดด้วยน้ำเสียงที่ให้เกียรติ บอกความต้องการของทีม/คนไข้ชัด ๆ เพื่อไม่ให้เกิดการ “ยอมเกินไปจนเหนื่อย”
A ต่ำ / ตรง ๆสื่อสารสั้น กระชับ อิงข้อเท็จจริง แยกประเด็นคนออกจากประเด็นงาน
ไวต่อความเครียด (N สูง)ลดความคลุมเครือ ยืนยันคำสั่งและบทบาทหลังส่งเวร (closed-loop) หลีกเลี่ยงการวิจารณ์กะทันหันโดยไม่มีบริบท

งานวิจัยด้านองค์กรพบว่า ความเห็นแก่ผู้อื่น และ ความไวต่อความเครียด เกี่ยวข้องกับการที่บางคน “รับรู้” ความขัดแย้งในที่ทำงานมากกว่าคนอื่น — ดังนั้นเหตุการณ์เดียวกันอาจถูกตีความต่างกัน; การสะท้อนกลับและขอดุลยภาพร่วมกันจึงสำคัญ

ในวอร์ด โรงพยาบาล และทีมพยาบาล — สิ่งที่งานวิจัยมักพูดถึง

  • ผลการปฏิบัติงานและพฤติกรรมองค์กร: มีงานวิจัยในบริบทพยาบาลที่พบว่าบุคลิกภาพเชื่อมโยงกับผลการทำงานผ่านพฤติกรรมเชิงบวกต่อองค์กร (เช่น การช่วยเหลือนอกเหนือหน้าที่) — เน้นว่า สภาพแวดล้อมและระบบสนับสนุน ยังมีบทบาทใหญ่
  • สุขภาวะการทำงานและการรับมือ: แนวโน้มที่พบบ่อยคือ E, C, A, O มีความสัมพันธ์เชิงบวกกับความเป็นอยู่ที่ดีในงาน ส่วน N สูงมักสัมพันธ์กับความเหนื่อยล้าทางอารมณ์มากขึ้น — โดย รูปแบบการรับมือ (เช่น การแก้ปัญหาเชิงรุก การปรับมุมมอง) มักเป็นตัวกลางสำคัญระหว่างบุคลิกกับความรู้สึกดีในงาน
  • การมีส่วนร่วม (engagement): ความเห็นอกเห็นใจเชิงรู้คิดและอารมณ์เชิงบวกมักเกี่ยวข้องกับความตั้งใจในงาน; บุคลิกเป็นเพียงชิ้นหนึ่งของภาพรวม
  • ความขัดแย้งในทีมพยาบาล: มีการศึกษาความขัดแย้งระหว่างพยาบาลร่วมกับบุคลิกภาพ — ใช้เป็นเหตุผลให้ฝึก ข้อตกลงการคุยกัน การส่งมอบด้วย SBAR และการสะท้อนหลังเหตุการณ์ แทนการตีความเป็นเรื่อง “นิสัยไม่ดี”
  • ความเหนื่อยล้าอาชีพ (burnout): งานวิจัยหลายชุดพบว่า N สูงและรูปแบบบุคลิกที่ “ไวต่อความเครียด” มักเกี่ยวข้องกับความเสี่ยงต่อ burnout มากขึ้น ในขณะที่ C, A, E, O ในระดับที่ช่วยสนับสนุนตนเองและทีมมักเป็นปัจจัยป้องกันบางส่วน — ไม่ใช่คำพยากรณ์เป็นรายบุคคล เพราะสภาพงานและทรัพยากรระบบมีผลแรงมาก

นำไปใช้ในภาคปฏิบัติ: การส่งเวรที่ชัดเด่นช่วยทุกสไตล์ — C ต้องการข้อมูลครบและลำดับ การเปลี่ยนแปลงในระบบต้องอธิบายให้ O และทีมเข้าใจ A ต้องมีพื้นที่พูดเรื่องยากโดยไม่ถูกมองว่า “สร้างเรื่อง” และผู้ที่ไวต่อความเครียดได้ประโยชน์จากความคาดหวังที่ชัดและการยืนยันกลับ (read-back) หลังคำสั่งสำคัญ

ข้อจำกัดสำคัญ

  • แบบประเมินนี้เป็น การรายงานด้วยตนเอง ในช่วงเวลาหนึ่ง ไม่ได้วัดความสามารถทางคลินิกหรือความเหมาะสมกับอาชีพ
  • วัฒนธรรมและภาษา มีผลต่อการตอบและการตีความ — หลีกเลี่ยงการยกสไตล์ใดสไตล์หนึ่งเป็น “แบบฝึกของพยาบาลที่ดี”
  • ใช้เพื่อ การพัฒนาตน การสื่อสารในทีม และการออกแบบการสนับสนุน ไม่ใช่การติดป้ายหรือตัดสินผู้อื่น

คำอธิบาย 4 มิติ + แนวทางใช้ในงานพยาบาล

D (Dominance) = กระทิง

ภาพรวม: ตรงไปตรงมา กล้าตัดสินใจ และมุ่งผลลัพธ์ชัดเจน เหมือน “นักรบ” ที่พร้อมเผชิญปัญหา.

พฤติกรรมเด่น: ชอบลงมือเอง เคลื่อนไหวเร็ว ชอบเห็นผลเร็ว กล้าเผชิญหน้าและแสดงจุดยืน.

ข้อดี: เด็ดเดี่ยว ปกป้องทีม กล้ารับผิดชอบในสถานการณ์กดดัน.

ข้อควรพัฒนา: ชะลอการตัดสินใจในบางเรื่อง รับฟังมากขึ้น และทบทวนก่อนสั่งการเพื่อไม่ให้ดูใช้อำนาจเกินไป.

I (Influence) = อินทรี

ภาพรวม: เข้าสังคมเก่ง ช่างพูด มีพลังบวก และถนัดสร้างบรรยากาศที่ดีให้ทีม.

พฤติกรรมเด่น: มองภาพใหญ่ คิดสร้างสรรค์ ชอบไอเดียใหม่ และเชื่อมโยงคนเก่ง.

ข้อดี: สร้างแรงจูงใจและความร่วมมือได้ดี ทำให้งานมีสีสันและคล่องตัว.

ข้อควรพัฒนา: ฝึกจัดลำดับความสำคัญ เก็บรายละเอียด และโฟกัสงานให้จบเป็นเรื่อง ๆ.

S (Steadiness) = หนู

ภาพรวม: สุภาพ อบอุ่น ใจเย็น และให้ความสำคัญกับความสัมพันธ์และความรู้สึกของคนรอบตัว.

พฤติกรรมเด่น: ชอบช่วยเหลือ ประสานงานเก่ง รับฟังดี และหลีกเลี่ยงความขัดแย้งรุนแรง.

ข้อดี: เป็นกาวใจของทีม ช่วยลดแรงปะทะ ทำให้ทีมทำงานร่วมกันได้ราบรื่น.

ข้อควรพัฒนา: ฝึกยืนยันจุดยืน กล้าตัดสินใจในบทบาทผู้นำ และตั้งขอบเขตเมื่อถูกกดดัน.

C (Conscientiousness) = หมี

ภาพรวม: ช่างวิเคราะห์ มีหลักการสูง รอบคอบ และให้คุณค่ากับความถูกต้องเป็นระบบ.

พฤติกรรมเด่น: ชอบวางแผนเป็นขั้นตอน ใช้เหตุผลสูง คิดล่วงหน้า และไม่ชอบความเสี่ยงที่ไม่จำเป็น.

ข้อดี: ลดความผิดพลาดได้ดี เหมาะกับงานที่ต้องการมาตรฐานและความแม่นยำ.

ข้อควรพัฒนา: เพิ่มความยืดหยุ่นเมื่อต้องเปลี่ยนแผน และลดความเคร่งเครียดจากความเป๊ะเกินไป.

การแปลผลแบบคู่สไตล์ (Top 2)

  • D/i: มุ่งผล + โน้มน้าวเก่ง ระวังพูดเร็ว/แรง
  • i/D: เข้าหาคน + กล้าชน ระวังใจร้อน
  • D/C: เด็ดขาด + มาตรฐานสูง ระวังเคร่ง
  • C/D: วิเคราะห์ + กล้าตัดสิน ระวังช้าเมื่อข้อมูลไม่ครบ
  • D/S: เร็วแต่ยังคุมความนิ่งได้ ระวังทีมสับสน
  • S/D: ใจเย็นแต่ยืนหลักได้ ระวังเก็บอารมณ์
  • i/S: เป็นมิตร อบอุ่น ประสานทีมเก่ง ระวังเลี่ยงขัดแย้ง
  • S/i: สุขุมแต่เข้าถึงง่าย ระวังไม่กล้าพูดสิ่งยาก
  • i/C: สื่อสารดี + มีเหตุผล ระวังเครียดเมื่อรายละเอียดไม่พร้อม
  • C/i: ข้อมูลแน่น + คุยรู้เรื่อง ระวังวิจารณ์มากไป
  • S/C: มั่นคง + รอบคอบ ระวังช้าเมื่อมีแรงกดดัน
  • C/S: ระเบียบ + ใจเย็น ระวังต้านการเปลี่ยนแปลง

ทำความเข้าใจโมเดล DISC

แบบย่อ

เหมาะกับเวิร์กช็อประยะสั้น ให้ผลลัพธ์รวดเร็ว โดยระบุสไตล์หลัก (และอาจมีสไตล์รอง) เช่น “คุณเป็นสไตล์ i และมีแนวโน้ม S”

แบบละเอียด

แสดงคะแนนครบทั้ง 4 มิติ พร้อมเปอร์เซ็นต์หรือกราฟสรุป เพื่อวิเคราะห์ความผสมของบุคลิก อย่างละเอียด เช่น “SC” หรือ “DI”

D: เด็ดขาด ผลลัพธ์นำ i: เข้าสังคม สร้างพลังทีม S: มั่นคง ใส่ใจคน C: รอบคอบ มาตรฐานสูง

เหตุใด DISC จึงสำคัญกับพยาบาล

งานพยาบาลต้องทำงานเป็นทีม ภายใต้ความกดดัน และกับผู้ป่วยที่หลากหลาย DISC ช่วยให้เข้าใจตนเองและผู้อื่น ปรับการสื่อสารให้ตรงคน ลดความขัดแย้ง ทำงานราบรื่นขึ้น และยกระดับคุณภาพการดูแลผู้ป่วย

4 มิติของ DISC

  • D(หมี) – มุ่งผลลัพธ์ ตัดสินใจเร็ว เด็ดขาด ตรงประเด็น แก้ปัญหาไว เหมาะสถานการณ์เร่งด่วน
    ระวัง: อาจตรงหรือกดดันคนอื่นเกินไป
  • I(อินทรีย์) – มุ่งความสัมพันธ์และการจูงใจ สื่อสารดี สร้างบรรยากาศทีม โน้มน้าวเก่ง
    ระวัง: ไม่ชอบรายละเอียด เสียสมาธิง่าย
  • S(หนู) – มุ่งความร่วมมือและความสม่ำเสมอ ใจเย็น รับฟังดี ดูแลต่อเนื่อง สร้างความมั่นคงให้ทีม
    ระวัง: ไม่ชอบความขัดแย้ง ตัดสินใจช้าเมื่อถูกกดดัน
  • C(หมี) – มุ่งความถูกต้องและคุณภาพ ละเอียด เป็นระบบ วิเคราะห์ดี ลดความผิดพลาด
    ระวัง: กลัวผิดพลาด สื่อสารน้อย ดูเข้มงวด

คู่สไตล์ — คนส่วนใหญ่ไม่ใช่แบบเดียว

  • D(หมี)I(อินทรีย์) – กล้าตัดสินใจ + จูงใจเก่ง พลังสูง นำทีมดี แต่ไม่ชอบรายละเอียด
  • S(หนู)C(หมี) – สุขุม + ละเอียด ทำงานรอบคอบ เสถียร แต่ปรับตัวช้าเมื่อเปลี่ยนเร็ว
  • iS(หนู) – เป็นมิตร อดทน ทำงานกับคนดี
  • C(หมี)D(หมี) – จริงจัง ละเอียด ตัดสินใจบนข้อมูล

การดู “คู่สไตล์” ทำให้เข้าใจคนลึกกว่า ไม่เหมารวม

ใช้ DISC ในงานพยาบาล (ภาคปฏิบัติ)

วอร์ดทั่วไป
D(หมี) ตัดสินใจเร็วในเหตุฉุกเฉิน, C(หมี) ตรวจความถูกต้องยา/อุปกรณ์, I(อินทรีย์) สื่อสารกับญาติ/ทีม, S(หนู) ดูแลผู้ป่วยใกล้ชิด
ผล: แบ่งบทบาทตามจุดแข็ง ลดซ้ำซ้อน ลดขัดแย้ง

หน่วยเฉพาะทาง / ห้องผ่าตัด
C(หมี) คุมมาตรฐานและความปลอดภัย, D(หมี) คุมเวลาและสถานการณ์, S(หนู) ติดตามผู้ป่วยต่อเนื่อง, I(อินทรีย์) สร้างแรงจูงใจและประสานทีม
ผล: งานแม่นยำ ทีมเข้าใจกัน ความเครียดลดลง

I(อินทรีย์)C(หมี)U
ต้องสมดุลทุกสไตล์: D(หมี) ตัดสินใจ, C(หมี) ป้องกันความผิดพลาด, S(หนู) ดูแลเชิงมนุษย์, I(อินทรีย์) รักษาการสื่อสารทีม
ผล: ตัดสินใจดี + ปลอดภัย + ทีมยังทำงานได้แม้เครียดสูง

บริหารและผู้นำ
รู้สไตล์ตน → รู้จุดแข็ง/จุดเสี่ยง
มอบงานให้ตรงคน สื่อสารให้ตรงสไตล์ → ทีมทำงานง่ายขึ้น
ใช้ D(หมี)I(อินทรีย์)S(หนู)C(หมี) แก้ความขัดแย้งและพัฒนาผู้นำรุ่นใหม่ได้จริง

ประโยชน์หลัก

  • สื่อสารดีขึ้น ทีมเวิร์กดีขึ้น
  • ดูแลผู้ป่วยตรงความต้องการมากขึ้น
  • พัฒนาบุคลากรและผู้นำ
  • ลดความเครียดและความขัดแย้ง

ข้อจำกัด

  • ไม่ใช่วัดความสามารถ และไม่ควรติดป้ายคน
  • ผลขึ้นกับการตอบและการตีความ
  • ใช้ผิดอาจกลายเป็นอคติ
  • ต้องคำนึงบริบทวัฒนธรรม
  • เป็น “เครื่องมือหนึ่ง” ไม่ใช่ทั้งหมด

การนำผลลัพธ์ DISC ไปใช้พัฒนาทีมงานพยาบาลในวอร์ด

การนำผลลัพธ์จากแบบประเมิน DISC (ในบริบทสัตว์ 4 ทิศ: กระทิง, อินทรี, หนู, หมี) ไปใช้พัฒนาทีมงานพยาบาลในวอร์ด สามารถทำได้โดยเน้นการ "ใช้คนให้ถูกงาน" และ "สื่อสารให้ถูกจริต" ซึ่งจะช่วยลดความขัดแย้งและเพิ่มประสิทธิภาพการดูแลผู้ป่วยได้จริง

1) การจัดมอบหมายงานให้ตรงกับจุดแข็ง

การรู้ว่าพยาบาลแต่ละคนเป็น “สัตว์” ชนิดไหน ช่วยให้หัวหน้าหอผู้ป่วยมอบหมายบทบาทได้เหมาะสมยิ่งขึ้น:

  • 🐂 กลุ่มกระทิง (D - Dominance)
    งานที่เหมาะสม: ผู้นำทีม (Team Leader) ในเวรดึก/ภาวะวิกฤต เช่น Code Blue, งานที่ต้องการผลลัพธ์รวดเร็ว เช่น บริหารจัดการเตียง หรือเคลียร์ข้อร้องเรียนที่ต้องการความเด็ดขาด
    การพัฒนา: ตั้งเป้าหมายท้าทาย แต่เตือนเรื่องความละเอียดอ่อนทางอารมณ์กับเพื่อนร่วมงาน
  • 🦅 กลุ่มอินทรี (I - Influence)
    งานที่เหมาะสม: งานประชาสัมพันธ์, กิจกรรมสันทนาการในวอร์ด, ผู้นำกลุ่มบำบัด/สอนสุขศึกษา เพราะสื่อสารโน้มน้าวและสร้างบรรยากาศบวกได้ดี
    การพัฒนา: ฝึกลงรายละเอียดและติดตามงานให้จบ เพื่อลดปัญหาเริ่มเก่งแต่ปิดงานยาก
  • 🐭 กลุ่มหนู (S - Steadiness)
    งานที่เหมาะสม: พี่เลี้ยง (Preceptor), ดูแลผู้ป่วย Palliative, งาน Routine/เอกสารที่ต้องการความสม่ำเสมอสูง
    การพัฒนา: สนับสนุนให้กล้าแสดงความคิดเห็น และระวังภาระงานเกินตัว (Burnout)
  • 🧸 กลุ่มหมี (C - Compliance/Conscientiousness)
    งานที่เหมาะสม: ประกันคุณภาพ (HA/QA), ควบคุมการติดเชื้อ (IC), ตรวจสอบคำสั่งการรักษาและยา
    การพัฒนา: ลดความเครียดจาก Perfectionism และฝึกยืดหยุ่นในสถานการณ์ที่ไม่วิกฤต

2) การปรับวิธีการสื่อสารในทีม

ปัญหาหลักในวอร์ดมักมาจาก “สื่อสารผิดรูปแบบ” การเข้าใจ DISC จะช่วยสร้าง ภาษากลาง ในทีม:

  • คุยกับกระทิง (D): “สั้น กระชับ ตรงประเด็น” — บอกปัญหา สิ่งที่ต้องทำ และผลลัพธ์ที่ต้องการ (เช่น SBAR)
  • คุยกับอินทรี (I): “เป็นกันเอง เปิดพื้นที่ให้แสดงไอเดีย” — เริ่มด้วยบรรยากาศผ่อนคลาย แล้วค่อยเข้าเรื่องงาน
  • คุยกับหนู (S): “นุ่มนวล เป็นขั้นตอน ให้ความมั่นใจ” — ไม่เร่งรัดหรือกดดันเกินจำเป็น
  • คุยกับหมี (C): “ข้อมูลพร้อม อ้างอิงหลักการ” — ใช้ข้อเท็จจริงและให้เวลาคิดวิเคราะห์ก่อนตอบ

3) การบริหารความขัดแย้ง (Conflict Resolution)

  • กระทิง vs หมี: ขัดกันเรื่อง “ความเร็ว” vs “ความละเอียด” → ทางแก้: ตกลง Deadline และมาตรฐานขั้นต่ำร่วมกัน
  • กระทิง vs กระทิง: แย่งบทนำ/แย่งกันพูด → ทางแก้: แบ่งขอบเขตอำนาจหน้าที่ (Zone) ให้ชัดเจน

4) การสร้างสมดุลทีม

  • หนูเยอะเกินไป: ทีมสงบและกลมเกลียว แต่เปลี่ยนแปลงยาก/ตัดสินใจวิกฤตช้า
  • กระทิงเยอะเกินไป: งานเร็วมาก แต่บรรยากาศตึงเครียดและโต้เถียงบ่อย
  • แนวทาง: จัดเวรให้คละสไตล์ เช่น มีกระทิงนำทีม มีหนูประสานงาน และมีหมีตรวจสอบยา

สรุป

การใช้ DISC ในวอร์ดไม่ใช่การตีตราว่าใครเป็นคนอย่างไร แต่เพื่อเข้าใจ แรงจูงใจ และ ความกลัว ของสมาชิกทีม เพื่อให้หัวหน้าดูแลใจและใช้งานทีมได้อย่างมีประสิทธิภาพสูงสุด
